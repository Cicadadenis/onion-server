## 📦 Установка
1. Скачайте скрипт `tor_host.sh` и сделайте его исполняемым: `chmod +x tor_host.sh`
2. Запустите скрипт: `./tor_host.sh`
3. Выберите `1` для установки сервера и необходимых пакетов
4. Выберите `2` для создания onion сайта
5. Выберите `3` для запуска сервера и Tor
6. Файловый менеджер будет доступен через ваш onion URL на порту `8081`

## ⚙️ Возможности
- Создание одного onion сайта
- Запуск PHP сервера на локальном хосте
- Автоматическая настройка Tor hidden service
- Web-панель с доступом к файловому менеджеру
- Отображение логина и пароля для файлового менеджера
- Бэкапы сайта

## 🚀 Использование
После запуска скрипта следуйте меню для создания и запуска сайта.

## 📝 Примечания
- Onion сайт доступен только через Tor Browser.
- Файловый менеджер защищён логином и паролем.
